# プロジェクト名の由来
Eigen + Weg (ドイツ語) = own + path (英語)

アイゲンヴェグと読みます．
日本語だと「自身の道（直訳）」や「唯一の解法（意訳）」といった訳になると思います．
Eigenはeigenvalue（固有値），eigenvector（固有ベクトル）と掛けてます．
